# 📄 关于 / About

## 🧾 简介

我是一名初一学生，就读于 NFLS（南京外国语学校）。

这里是个很自由的地方（尽管其他人还是很卷），但我终于可以躺平了。

至于我的爱好嘛，很多：编程、乒乓球、画画、弹琴、魔方、下棋……
可悲的是，后面几项等到初中后就无影无踪，只剩下乒乓球和编程参加了学校社团。

## 🎓 我和学校

说到学校，就不可避免的问一句：我喜不喜欢学校？

嗯，挺喜欢的吧，虽然可能更喜欢的是见到同学。当然，我也不排斥上课这件事，学习本来就是一件很有乐趣的事。

上小学一开始我还是比较菜的，只是上了家旁边的一所公办学校。后来跟随小升初的潮流选择冲外。因为南外（也只有南外）是通过考试录取，凭实力，其他学校都是摇号靠运气。
所以南外确实一个很好的选择。最终我顺利考上，但不可免的是哪里群英荟萃，我成绩当然没有小学突出。

我却发现了一点初中和小学的不同，夸张点说，小学学习成绩的“阶级分化”十分严重，大部分都是学习好的一起玩。而中学呢，就没有这种现象（也许只是我的学校？），好像没人在意成绩。

我的成绩也不是最好的吧，但也能平均混进班级前五，希望能再接再厉（？）。

## 💻 我和编程

一开始学 Python，从未接触过“代码”的我曾认为 Python 是个很难的东西，但显然，事实并非如此。我很快对 Python 有了兴趣，然后就不上课了。毕竟学都能学会，还上课干嘛？

过了一年，我又开始学习 C++。接着我发现，C++ 似乎比 Python 难，而且难得不止一点。当时的我也懒得自学，就跟着上课刷题，也不干别的……吗？

为了寻找乐趣，我研究起图形库，EasyX。

> 众所周知，写游戏是学习任何语言的必经之路

也正是这时候我开始在 CSDN 上写[博客](https://blog.csdn.net/2201_75791084/)，都是些用 EasyX 写的小游戏，没想到还有挺多人看。

过了半年就不写了，因为学业压力大（小升初）。我也不知道那个暑假干了些什么，反正跟编程没有啥关系。

上了初中，情况有了些改变：
- 有了学校的社团和一周两节的信息课
- 学习压力相对变轻了（NFLS 是这样的）
- 会编程的同学变多了

这是个很好的消息，我可以和同学一起写代码了。

话说回 C++：痛苦的信息竞赛。

虽然我基本不在信息课上打游戏（？），但一般都在干别的事，反正不听课。我不喜欢抽象的东西，比如这些数据结构，一点儿也不直观（这也是为什么我还去学了 Unity，以及一些 UI 库）。

而且我一直不明白一件事：信息竞赛有什么用？能干嘛呢？

无非就是会一点算法，拿一个一等奖。每次看到那些大佬们又拿了什么什么奖，敬佩之余我还在想：他们真的会写代码吗？
至少我认为，算法竞赛就像是高等数学，虽然真的有人去学，但是几乎在生活中用不上。

> 截止至 2024.8 我还不明白 QAQ

正如上文所说，我过了半年又开始研究起 Unity。接着又是 Python 的一些杂七杂八的内容。

关于编程，就写到这吧~

## 🖥 关于网站
### 这个小站是怎么建成的？

我曾经考虑过搭建一个属于自己的个人网站，能在互联网上拥有自己的一片天地，这如何不是一件好事？
坏消息是，我只会 Python 和 C++，你说我该怎么办？

我开始寻找一些现成的建站工具，我甚至连前端是什么都不知道。直到后来我看到了别人的网站，很显然他们并没有用建站工具。
于是，我开始学习 HTML 和 CSS，尝试自己写一个网站。很快，连续写了好几个很丑的网站之后，我不想玩了。

虽然 HTML 和 CSS 十分简单的东西，但写起来很麻烦，而我又是一个很懒的人，不想写了，于是又开始寻找建站工具。
但这次不一样，我找的是框架，而不是工具。接着，我就找到了一样东西：Astro。

作为一个懒人，我自然是寻找了一个主题（也就是现在这个），直接拿来用。
我对这个主题很满意，就 clone 下来，稍微做修改内容，用了仅仅一天就搭建完成了，真的是懒人的福音。

我还发现一个问题，由于我一直觉得它很丑，所以一直在无休止的美化、修改CSS，这是一件很痛苦的事。我更无法安心地写起博客了。
而博客模板，真的完美解决了这个问题。我对它满意才选的这个模板，所以我就不会再陷入美化的陷阱中，可以安心地写博客了。
